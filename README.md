# calc-rs

Simple calculator implemented in Rust to learn the language a bit.

## Usage
`cargo run` in the directory

## Features
- Basic operations (`+`, `-`, `*`, `/`, `^`)
- Decimal numbers
- Variables (assigned with `:`)
  - Including "reserved" variables (`ANS`, `PI`, `RAND`)
- PEMDAS (and grouping with `(` & `)`)
- Comparators (`=`, `>`, `<`)
